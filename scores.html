Introduction

This document reads and summarizes the NBA historical games dataset from Games.csv using R and the tidyverse package suite.

Loading Libraries and Data

First, we load the tidyverse library and read in the dataset.

library(tidyverse)

# Read the NBA games dataset
games <- read_csv("Games.csv")
